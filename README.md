# html2docx

Generate docx from htm(l).

## HOW TO USE

Terminal

```
> html2docx.py
Dir: <Drag and drop the folder containing htm(l) files.>
Processing...
...
...
...
Done!
Enter to exit.
```

**NOTE:**

- After processing is complete, a copy (\_\_copy) folder is created. The Docx(s) will be generated in that folder.

## REQUIRES

- Windows
- Word 32bit
